# <h1 align="center"> FIAT II</h1>

### Install Foundry
```sh
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

### Set Environment
Copy and update contents from `example.env` to `.env`

## Building & Testing

```sh
git clone https://github.com/f1a7/fiat-ii
cd fiat-ii
make # This installs the project's dependencies.
make test
```
